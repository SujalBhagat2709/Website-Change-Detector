# Website Change Detector

## Overview

A tool that continuously monitors websites and alerts users whenever page content changes.

---

## Features

- Website monitoring
- Change detection
- Continuous tracking
- Real-time alerts

---

## Files

- monitor.py
- detector.py

---

## Installation

```bash
pip install requests
```

---

## Usage

```bash
python detector.py
```

---

## Example

```text
Enter Website URL:
https://example.com

Monitoring Started...

No Change Detected

No Change Detected

🚨 WEBSITE CHANGED!

Detected at 14:32:05
```
